## Project Direction

Default toward **Lyra AI**, a real Sui product with polished UX, meaningful Sui
usage, working on-chain evidence, protocol integrations, and a credible path
beyond the hackathon.

Lyra AI is a Sui-native autonomous finance agent that converts user goals into
policy-checked PTBs, executes only within approved protocol scope, and stores
auditable memory and receipts with Walrus.

## Hackathon Priorities

- Show why Sui is necessary, not incidental.
- Prefer working testnet or mainnet flows over mock-only demos.
- Prioritize sponsor protocols: Walrus/MemWal for memory and receipts, and
  DeepBook/DeepBook Predict for financial execution.
- Use Cetus/Turbos, NAVI/Suilend, and DefiLlama as supporting integrations only
  when they strengthen the sponsor-centered demo.
- Keep AI bounded by deterministic policy checks.
- Demonstrate successful execution and blocked unsafe behavior.
- Keep submission materials aligned with the Sui Overflow 2026 handbook.
